## Topic
The topic we have chosen is about evaluating the performance of various system of linear equation solvers for Alternating Least Squares (ALS) for matrix factorization in recommender system, particularly in the context of collaborative filtering. We will compare the performance of different solvers such as Matrix Inversion, LU Decomposition, Cholesky Decomposition, Partial Pivoting, and QR Decomposition in terms of accuracy, computational efficiency, and scalability. We will also analyze the impact of these solvers on the convergence rate of the ALS algorithm and the quality of recommendations generated by the matrix factorization model.

## Implementation
- Jupyter Notebook for code implementation and results visualization.
- All code for solvers are barebones implementations, only Numpy is used for basic matrix operations, no built-in solvers are used.
- The manuscript will be written in Latex.
- The dataset used for testing will be the MovieLens dataset, which is commonly used for evaluating recommender systems. We will use a subset of the dataset to ensure that our experiments are computationally feasible while still providing meaningful insights into the performance of the solvers.
